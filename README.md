# hicetnunc_explorer

simple tool to get an overview of a [hicetnunc.xyz](https://hicetnunc.xyz) collection

live site ...
## current features

visible on one page:
* objkt
* objkt's creator infos
* current swap list
* amount of tezos spent to collect the piece (with date)


## Project setup
```
yarn install
```
